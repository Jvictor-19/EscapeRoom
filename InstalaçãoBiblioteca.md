# Como instalar a biblioteca na Arduino IDE
1. Dentro da IDE siga o caminho File > Preferences... > Sketchbook location
2. O diretório presente dentro da caixa é onde ficará os arquivos, você pode escolher outro caminho clicando no botão à direita
3. Mova/copie os arquivos que você quer ter acesso da pasta Modules até o diretório do passo anterior
4. Inclua os arquivos no código usando ``` #include <modulo.hpp> ``` substituindo o "modulo" pelo nome do modulo que você quer incluir
